About Fraction Code

Fraction code is a code made for writing secret messages. It is not a programming language. Fraction code was made simply by cutting off a part of each letter. The best thing about fraction code is that it is easy to read and write off the top of your head without having to think hard about each individual letter in the code (and it's still pretty hard to decode!).

The Glyphs

Currently there are only the uppercase letter glyphs. More may be added sometime, but you really only need these letters to send messages in code. They are listed alphabetically, but you can also hover over the letters to learn what each letter is. Download all the SVG glyphs here.

(The color of the letters is a light gray so you can see it on black as well as white backgrounds.)
