Project: Menubars

One of my favorite parts of designing a website is creating a unique and beautiful menubar. I think the menubar defines the theme of the website, since basically everything else is the same. So here are some menubar ideas for any website! Note: all these links except for the Menubars link goes nowhere.


Simple

Whenever you hover over a page or on a page, it is bold.

Home About Blog Menubars

Rainbow

Each page in the menubar is a different color of the rainbow when you hover over it.

Home About Blog Menubars

Enlarge

Each link enlarges when you hover over it. I also made it change hue, but this is optional.

Home About Blog Menubars

Bullet Points

There is a bullet point between each link in the menubar, separating each page. Each page turns a different hue when you hover over it.
